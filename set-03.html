✦ 客戶服務
Q1
Our customer service team worked overtime to ------- all outstanding complaints before the end of the quarter.
【中譯】我們的客戶服務團隊加班工作，以便在季末前處理所有待處理的投訴。
A
resolve
B
address
C
handle
D
tackle
解題關鍵
address complaints 為客服情境最標準的搭配，表示「針對性地回應並處理投訴」。resolve 強調最終解決，不一定含回應之意；handle 較口語，且搭配 complaints 不如 address 正式；tackle 多用於面對複雜挑戰，不是 complaints 的慣用搭配。
陷阱提示
resolve 看似最合適，但 resolve complaints 強調問題被徹底解決，而題目重點在於「處理」動作本身；address complaints 才是正式商業書面中的慣用語。
單字/片語詞性音標主要意思搭配用法
resolve動詞/rɪˈzɒlv/解決（問題）resolve a dispute
address動詞/əˈdrɛs/處理；應對address complaints
handle動詞/ˈhændl/處置；管理handle a situation
tackle動詞/ˈtækl/應對（困難任務）tackle challenges
字詞家族
address
動詞・處理、應對（正式）
address concerns
片語・回應顧慮
address an issue
片語・處理某項問題
address the audience
片語・向聽眾致詞
Q2
Repeating the same hiring patterns tends to ＿＿ existing biases within the organization.
【中譯】重複相同的招募模式往往會__組織內既有的偏見。
A
sustain
B
perpetuate
C
reinforce
D
maintain
解題關鍵
perpetuate 特指使（負面循環、偏見、迷思等）長久延續，是描述系統性問題根深蒂固的精確用詞；reinforce 強調強化某信念；sustain 與 maintain 較中性，不帶「使不良狀況持續」的語義。
陷阱提示
perpetuate vs. reinforce：兩者均可搭配偏見，但 perpetuate 強調「使其在時間上持續存在」，reinforce 強調「使其更強固」；招募模式導致偏見延續 → perpetuate。
單字/片語詞性音標主要意思搭配用法
sustainv/səˈsteɪn/維持、支撐（長期）sustain growth / momentum
perpetuatev/pərˈpetʃueɪt/使長久延續（尤指不良狀況）perpetuate a myth / bias ✓
reinforcev/ˌriːɪnˈfɔːrs/強化、加強reinforce a stereotype / belief
maintainv/meɪnˈteɪn/維持、保持（一般）maintain standards / order
字詞家族
perpetuate
v. 使長久延續
perpetuation
n. 延續、使永久化
perpetual
adj. 持續不斷的
ingrained bias
ph. 根深蒂固的偏見
Q3
After joining the night shift, Jenny gradually ＿＿ working late hours and now finds it manageable.
【中譯】加入夜班後，Jenny 逐漸__晚工時，現在覺得尚能應對。
A
used to
B
got used to
C
was used to
D
would use to
解題關鍵
get used to + V-ing = 漸漸習慣（強調從陌生到適應的動態過程）；句中 gradually 強調過程 → got used to working；used to + V 表過去習慣（現已停止）；was used to + V-ing 表過去的靜態習慣狀態。
陷阱提示
used to + V（過去習慣，現已停止）vs. be/get used to + V-ing（習慣某事）：三者結構不同；get 強調動態轉變過程，be 強調已達到的靜態狀態。
文法補充
used to + V（過去習慣）/ be used to + V-ing（習慣某事，靜態）/ get used to + V-ing（漸漸習慣，動態）
單字/片語詞性音標主要意思搭配用法
used toph/juːst tuː/過去習慣（現已停止）used to + V → 過去曾經（現不再）
got used toph/ɡɒt juːst tuː/漸漸習慣了（動態過程）get used to + V-ing ✓
was used toph/wɒz juːst tuː/過去已習慣（靜態狀態）was used to + V-ing → 過去的習慣狀態
would use toph/wʊd juːs tuː/×（不存在的錯誤結構）would use to 為非標準錯誤用法
字詞家族
used to V
ph. 過去習慣（現已停止）
be used to V-ing
ph. 習慣於某事（靜態）
get used to V-ing
ph. 漸漸習慣（動態）
manageable
adj. 可應對的、可管理的
Q4
Not until the lights went out ＿＿ they realize that the power outage affected the entire neighborhood.
【中譯】直到燈光熄滅，他們才__意識到這次停電影響了整個社區。
A
did
B
had
C
would
D
could
解題關鍵
否定副詞片語 Not until... 置於句首時，主句須做倒裝：did + S + V，此為英文倒裝語法的標準結構。
陷阱提示
had 雖為過去式助動詞，但此句並非過去完成時態；would/could 為情態動詞，不適用於倒裝否定副詞句型中的陳述事實。
文法補充
否定副詞倒裝：Not until + 時間子句，did/does/do + S + V（主句語序翻轉，助動詞前移至主詞之前）
單字/片語詞性音標主要意思搭配用法
didaux. (inverted)/dɪd/（助動詞，倒裝語序）Not until... did + S + V → 否定副詞倒裝，助動詞 did 前移 ✓
hadaux. (past perfect)/hæd/（過去完成式助動詞）過去完成式 had 不符合本句時態（went out 為過去簡單式，無需完成式）
wouldaux. (conditional)/wʊd/（情態動詞，條件語氣）would 用於假設或意願，此句為陳述事實，語氣不符
couldaux. (ability/permission)/kʊd/（情態動詞，能力或允許）could 表能力或許可，與 realize（意識到）搭配語意不自然
